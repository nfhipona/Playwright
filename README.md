# TypeScript Programming, Playwright Web (End-To-End) , Accessibility, Visual and API Automation Testing & AI features

1. Run `serve ./server` to start and serve the single page example html page. Or, run `npm run start` to call in the provided script.
2. Run `npm run tests` to run the script which has `npx playwright test && npx playwright show-report` playwright script.
3. The playwright environment is configured to trigger `npm run start` to launch the single page html sample page.
