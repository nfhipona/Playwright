# Udemy Course: TypeScript Programming, Playwright Web (End-To-End) , Accessibility, Visual and API Automation Testing & AI features

# Instructions
1. Clone the repo
2. Install and update packages with: `npm install`
3. Run tests suite with: `npm run tests` or `npx playwright tests`

------

HTML file found in `server/index.html` is an attachment from the Udemy course that was configured to be served as a local page to be tested against with Playwright.

1. Run `serve ./server` to start and serve the single page example html page. Or, run `npm run start` to call in the provided script.
2. Run `npm run tests` to run the script which has `npx playwright test && npx playwright show-report` playwright script.
3. The playwright environment is configured to trigger `npm run start` to launch the single page server which is located at `server/` folder.

-------

To do unit tests on page: [saucedemo](https://www.saucedemo.com/)
